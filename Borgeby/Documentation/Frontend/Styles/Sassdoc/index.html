    breakpoints

    $breakpoints: (
    "tablet"                    "screen and (min-width: #{em($breakpoint-tablet-start)})",
    "laptop"                    "screen and (min-width: #{em($breakpoint-laptop-start)})",
    "desktop"                   "screen and (min-width: #{em($breakpoint-desktop-start)})",
);

    Description

    Breakpoint aliases and conditions (in em). These can be invoked via the media-query() mixin. We use a mobile first approach and therefore only use min-width and not max-width in conditions.

    Used by

    • [mixin] media-query

    widths-columns

    $widths-columns: (
    1,
    2,
    3,
    4,
    5,
    6,
);

    Description

    Grid size fractions for layout items. Add the fractions you want to use when sizing grids. E.g. if you add 6, you can use u-1/6, u-3/6, u-6/6, etc.

    Example

    <div class="layout__item  u-1/6"></div>

    Used by

    • [mixin] widths

    widths-columns-responsive

    $widths-columns-responsive: (
    1,
    2,
    3,
    4,
    5,
    6,
);

    Description

    Responsive grid size fractions for layout items. Add the fractions you want to use when sizing responsive grids. The fractions are then used in combination with the projects defined breakpoints. E.g. if you add 6, you can use u-1/6-tablet, u-3/6-laptop, etc.

    Example

    <div class="layout__item  u-1/6-tablet"></div>

    See

    • [variable] breakpoints

    z-indexes

    $z-indexes: (
    "modal":            5000,
    "navigation":       4000,
    "default":          1,
);

    Description

    Z-index map that contains all custom z-indexes. The z-indexes are used by calling the z-index() mixin.

    Used by

    • [mixin] z-index
